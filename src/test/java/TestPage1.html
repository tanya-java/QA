BMW X7 Description

The X7 is BMW's biggest SUV. A class up from the midsize X5, the X7 is a large luxury SUV that comes standard with three rows of seating (a third row is optional on the X5). ... The standard X7 xDrive40i is powered by a turbocharged 3.0-liter inline six-cylinder engine that produces 335 horsepower and 330 lb-ft of torque.
